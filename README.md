# Allar's Blog

This is the repo for my website http://www.allarsblog.com.

It is built using [Jekyll](https://jekyllrb.com/) on top of [Neo-HPSTR Jekyll Theme](https://github.com/aron-bordin/neo-hpstr-jekyll-theme).'

# Development Setup

1. Clone repo
1. `bundle install`
1. `jekyll serve`
